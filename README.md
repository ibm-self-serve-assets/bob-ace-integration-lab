# IBM Bob, ACE and IWHI Integration Lab

## Overview

This repository contains a comprehensive hands-on lab demonstrating how to expedite IBM App Connect Enterprise (ACE) integration projects development using IBM Bob AI-assisted development, combined with centralized management through IBM WebMethods Hybrid Integration (IWHI) Hybrid Control Plane.

## What You'll Learn

This lab showcases the powerful combination of:
- **IBM Bob**: AI-powered development assistant that accelerates ACE integration project creation through natural language prompts
- **IBM ACE**: Industry-leading integration platform for building enterprise integration solutions
- **IWHI Hybrid Control Plane**: Centralized management and deployment platform for hybrid integration environments

## Lab Highlights

- Create ACE integration servers on Linux VMs and manage them centrally with IWHI
- Import and use custom IBM Bob modes for ACE development
- Generate complete REST API integration projects using simple prompts
- Validate and test ACE projects locally using ACE Toolkit
- Deploy integration projects to managed runtimes through IWHI Hybrid Control Plane
